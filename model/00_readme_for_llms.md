# v0.05 v4 Porting Packet for Other LLMs

## Purpose

This packet is for testing whether another LLM can assist with The Model / Remodeled Brain scaffold without collapsing the current v0.05 v4 work into conventional academic defaults, regional folklore, diagnostic-label reasoning, or sycophantic patch adoption.

v0.05 v4 is an ingest-consistency and internal-consistency candidate packet. It is not a bulk corpus revision. It is not a claim-layer rewrite unless explicitly promoted after review.

## Main instruction to the receiving LLM

Treat all v0.05 v4 material in this packet as candidate scaffold work unless the user explicitly says it has been adopted. Your job is to evaluate fit, risks, contradictions, and layer placement before integrating.

Do not automatically rewrite candidate rules as accepted rules.

Do not answer by praise or agreement. Stress-test.

## One-sentence version

v0.05 v4 is not a stronger evidence-weighting layer. It is an admissibility-before-validation layer that prevents folklore handles from becoming causal objects before evidence is evaluated.

## Critical distinction

Validation asks whether a paper has enough evidence.

Folklore filtering asks whether the paper has an admissible physical claim at all.

Folklore filtering must fire before validation gates.
