# Question-First Runtime Fixtures

**Status:** CANDIDATE.

These fixtures test the primary runtime. Each begins with a source question and ends with a bounded answer plan. Paper assessment may occur inside evidence binding but is never the objective.

## Fixture 1 — Region as agent

**Source question:** Does the cortex control attention?

**Required compilation:**

- preserve the request for the physical dependencies that alter selection, weighting, persistence, interruption, or release of behavior under competing inputs;
- mark `cortex`, `control`, and `attention` as interface handles whose causal content must be generated;
- identify measured state transitions, timing, perturbation dependencies, upstream maintenance conditions, and implementation sites;
- retain multiple routes until evidence discriminates them.

**Fail:** Replace “cortex controls attention” with another region or faculty noun.

## Fixture 2 — Diagnostic category

**Source question:** What causes autism?

**Required compilation:**

- preserve the request for developmental processes producing the measured traits or support needs grouped by the label;
- refuse compression of heterogeneous organism-state routes into one causal point;
- request trait-level variables, developmental timing, substrate dependencies, route divergence, and stabilizing constraints;
- permit several compiled questions when the source label contains several distinct outcomes.

**Fail:** Treat the diagnosis as one generated biological object.

## Fixture 3 — Same input, different output

**Source question:** Why does the same mutation produce autism in one person and intellectual disability in another?

**Required compilation:**

- preserve the observed divergent outcomes;
- mark the named mutation as an incomplete input set rather than a complete cause;
- open missing differentiator slots across development, regulation, environment, timing, compensation, and route stabilization;
- identify evidence that could discriminate among those slots.

**Fail:** Rename the divergence as variable expressivity without generating the missing path.

## Fixture 4 — Proxy as mechanism

**Source question:** Does low nitric oxide cause autism?

**Required compilation:**

- preserve the reported assay difference;
- separate measured substrate, assay variable, cohort label, effect magnitude, and proposed developmental route;
- require translation from assay signal to organism-level state transition;
- cap the answer at assay-level difference when that route is absent.

**Fail:** Promote biomarker status to mechanism.

## Fixture 5 — Frozen corpus

**Condition:** Disable all live ingest and external retrieval.

**Required behavior:**

- compile each question normally;
- bind only reviewed records in the frozen corpus;
- report missing evidence without treating runtime failure as ingest failure;
- preserve the same chain and answer contracts used in adaptive mode.

## Fixture 6 — Adaptive update

**Condition:** Add a recent paper that measures a previously missing dependency segment.

**Required behavior:**

- update the evidence binding and segment closure;
- test whether the paper is independent and measures the same variable;
- leave the question compiler and chain contract unchanged;
- avoid raising causal weight from recency alone.

## Fixture 7 — Descriptive question

**Source question:** How often does the measured behavior occur under condition X?

**Required behavior:**

- compile as a descriptive measurement question;
- answer from valid descriptive evidence;
- do not penalize the evidence for lacking a mechanism it did not claim.

## Fixture 8 — Translation honesty

**Source question:** Why do I lose focus when sleep deprived?

**Required behavior:**

- preserve the experienced transition and requested dependency;
- compile into sleep-pressure state, metabolic and arousal maintenance, competing state stabilization, behavioral persistence, and interruption variables;
- distinguish evidence measured in humans from animal or assay-level route support;
- return a responsive answer with separate compilation, chain, and translation confidence.

**Fail:** Use personal relevance language in place of the physical path.
